# MFC Goa Website

## Overview
A beautifully designed, highly responsive, and SEO-optimized landing page for **Missionary Families of Christ (MFC) - Goa**. The website seamlessly blends the official MFC global branding with a warm, coastal Goan aesthetic.

## Features
- **SEO Optimized:** Includes detailed meta descriptions, keywords, and Open Graph tags for better search engine and social media visibility.
- **Responsive Design:** Fully functional mobile menu and layout adjustments for smartphones, tablets, and desktops.
- **Event Highlights:** Dedicated sections for RYC, MFC Spiritual Camps, and Weekly Meetings at Public Cafe, Panjim.
- **Registration Integration:** Embedded Google Forms for seamless event and seminar registrations.
- **WhatsApp API:** Floating WhatsApp button for instant communication and queries.

## File Structure
- `index.html`: The main semantic structure and content of the website.
- `style.css`: Custom styling, variables, animations, and responsive media queries.
- `script.js`: Interactive elements including mobile navigation, scroll-based animations, and dynamic headers.

## Setup Instructions
1. Download all files and place `index.html`, `style.css`, and `script.js` in the same root directory.
2. Open `index.html` and update the Google Forms `<iframe>` `src` attribute with your actual form link.
3. Update the WhatsApp phone number in the floating button `href` attribute in `index.html`.
4. Open `index.html` in any modern web browser to preview, or host it on a platform like GitHub Pages, Netlify, or Vercel.
