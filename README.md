# MFC Goa Web Architecture Landing Page[span_0](start_span)[span_0](end_span)

## Overview
An interactive, responsive, and search-engine-optimized Single Page Web interface engineered explicitly for **Missionary Families of Christ (MFC) - Goa**[span_1](start_span)[span_1](end_span). The site balances the official values framework of the Global MFC ecosystem while celebrating the high-energy coastal aesthetic of Goa.

## Features
- **Dynamic Dark/Light Mode Switch:** Implements cross-session state persistence powered entirely by `localStorage`.
- **Comprehensive Mobile-First Viewports:** Smooth navigation menu transitions and layout collapsing customized for cross-browser device scaling.
- **Optimized OpenGraph & Meta Assets:** Pre-configured structural schema tags designed to improve indexing visibility and social media preview formats.
- **Expandable Dynamic WhatsApp Floating Interface:** A dual-action communication widget linking users cleanly with community leaders:
  - **Tito Gladwin** (`+91 98812 47783`)
  - **Tita Josephine** (`+91 99237 37783`)
- **Action Embed Layer:** Seamless inline mapping integration for event registrations through Google Forms architectures.

## Production Assets Included
1. `index.html` - Clean structural layout featuring accessible typography configurations and layout schemas.
2. `style.css` - Custom animations, global UI transitions, and dark-theme design override blocks.
3. `script.js` - Logic controlling theme storage pipelines, mobile toggle states, and DOM scroll reveal triggers.

## Configuration Deployment Setup
1. Keep `index.html`, `style.css`, and `script.js` stored in the same central folder repository directory.
2. Open `index.html`, search for the string `https://docs.google.com/forms/...` and change out the value placeholder with your active programmatic Google Forms destination reference string link.
3. Serve using any static deployment solution engine (e.g. GitHub Pages, Netlify, Vercel, or traditional FTP environments).
